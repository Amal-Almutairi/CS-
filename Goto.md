# كلمة الـ Goto :

يستخدم للانتقال في البرنامج من قسم الى اخر 

##  القيود
-  يجب أن يكون الوسم الهدف في نفس الملف ونفس السياق 
- لا يمكن  الانتقال إلى أي نوع من أنواع الحلقات أو إلى بنية switch

 ## مثال: 
 
<div dir=ltr>

```
int x = 1;

        Loop:

            x++;

            if (x < 100)

            {

                goto loop;

            }
```
