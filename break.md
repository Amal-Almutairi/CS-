# كلمة الـ Break :

يوفِّر لك التعبير break القدرة على الخروج من حلقة التكرار عند حدوث عامل خارجي

 ## مثال: 
 
<div dir=ltr>

```

number = 0

for number in range(10):
   number = number + 1

   if number == 5:
      break    # break here

   print('Number is ' + str(number))

print('Out of loop')

```
